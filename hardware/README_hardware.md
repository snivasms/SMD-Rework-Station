# Hardware Design Files

This folder contains schematic and PCB fabrication files for the
SMD Rework Station project.

Processor Board:
- ESP32 controller
- MAX6675 thermocouple interface
- safety interlocks
- fan control

Power Board:
- transformer power supply
- triac heater control
- fan boost converter
- 5V and 12V regulators

PCB files include:

layout files – for reference  
negative copper layers – for PCB etching or toner transfer  
mirrored top layer – for top copper fabrication

Prototype was made by negative photo resist methord.
